## NOTAS IMPORTANTES

### Formato de Bloques de Código

- CADA campo de contenido debe estar en su propio bloque de código ```
- Los títulos de sección se mantienen como markdown normal
- Los nombres de campos en **negrita** se mantienen fuera de los bloques
- Esto permite copiar directamente cada contenido al formulario web

### Formato de Texto Dentro de los Bloques

- **Negrita**: `**texto**` para conceptos clave
- _Cursiva_: `*texto*` para énfasis
- **_Negrita+Cursiva_**: `***texto***` para información crítica
- Saltos de párrafo: `\\n` (excepto en JSON de recomendaciones donde es `\n`)

### Validación

- Todos los campos obligatorios deben completarse
- JSONs deben estar correctamente formateados
- Extensión total: 1500-2500 palabras
- Formato aplicado estratégicamente (máx 20% del texto)
